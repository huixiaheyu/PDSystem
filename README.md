# 拼豆店计时系统

一个基于 Python + PyQt6 的本地桌面计时工具，适用于拼豆店、手作店、桌游店等需要同时管理多桌倒计时的现场场景。

## 功能特点

- 支持多桌同时计时
- 每桌独立设置预设时长
- 每桌支持通过弹窗自定义加时（小时 / 分钟）
- 开始后显示预设时间与大字号倒计时
- 支持三种运行状态：进行中、即将到时、已超时
- 状态变化会通过颜色、状态徽标和卡片高亮进行区分
- `截止` 与 `设置保存` 使用统一风格的确认弹窗
- 顶部支持展示桌位总数、进行中、即将到时、已超时统计
- 支持切换全屏显示
- 默认参数使用 `QSettings` 自动保存，关闭软件后重新打开不会重置
- 提供按天文本日志，便于追溯现场操作和问题排查

## 运行环境

- Python 3.10+
- PyQt6

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动方式

```bash
python3 main.py
```

## 使用说明

### 主界面

主界面展示所有桌位卡片，每一行包含：

- 桌号
- 当前状态
- 预设时间
- 倒计时
- 时间输入框
- `加时` 按钮
- `开始 / 截止` 按钮

交互规则：

- 空闲时可直接修改该桌的小时和分钟
- 点击 `开始` 后进入计时
- 开始后 `加时` 按钮变为可用，点击后会弹出加时窗口
- 点击 `截止` 会弹出确认窗口，确认后结束该桌计时
- 倒计时区域会横向弹性伸缩，按钮保持固定宽度

### 设置

点击顶部 `设置` 按钮可以修改：

- 总桌数
- 默认小时
- 默认分钟
- 提前提醒（分钟）

点击 `保存` 后：

- 会先出现确认弹窗
- 新配置会立即生效
- 配置会自动持久化保存
- 下次打开软件时继续使用上次设置

### 状态说明

- 空闲：未开始计时，可编辑时间
- 进行中：正常倒计时中
- 即将到时：剩余时间进入提醒阈值内
- 已超时：倒计时已小于 0

### 日志功能

程序会自动记录关键事件日志，日志按天写入文本文件。

当前已记录内容包括：

- 应用启动
- 应用退出
- 开始计时
- 截止计时
- 加时
- 设置变更
- 全屏切换
- 进入“即将到时”状态
- 进入“已超时”状态
- 无效操作（例如 0 分钟开始、0 分钟加时）

当前日志目录：

```text
PD-System-logs/
```

日志文件示例：

```text
PD-System-logs/2026-04-13.log
```

## 配置持久化

当前项目使用 `QSettings` 保存以下配置：

- `table_count`
- `default_hours`
- `default_minutes`
- `warning_minutes`
- `start_fullscreen`

这意味着你不需要每次启动软件都重新设置默认参数。

## 打包说明

项目依赖中已包含 `PyInstaller`，后续可在 Windows 环境下打包为可执行文件。

示例命令：

```bash
pyinstaller --noconfirm --windowed --name "拼豆店计时系统" main.py
```
